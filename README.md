# Superstore-Sales-EDA
Exploratory Data Analysis of Superstore sales data using Python, pandas, and matplotlib

## Tools Used
- Python
- pandas
- matplotlib

## Project Objectives
- Clean and inspect raw sales data
- Analyze sales by region and category
- Identify top-performing states
- Create visual dashboards

## Key Insights
- West region had highest sales
- Technology category generated highest revenue
- California was top-performing state

## Files
- eda.py : analysis code
- train.csv : dataset
